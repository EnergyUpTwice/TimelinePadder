1. 用 Visual Studio 打开 .csproj
2. 编译目标选 Release / Any CPU
3. 把生成的 TimelinePadder.dll
   复制到：
   %APPDATA%\Subtitle Edit\Plugins\
   或
   <SE安装目录>\Plugins\
4. 重启 Subtitle Edit
5. 菜单 Synchronization (同步) → Timeline Padder – 时间轴智能扩展
